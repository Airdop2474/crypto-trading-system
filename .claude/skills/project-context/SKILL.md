# project-context

Quick project context and navigation aid.

## Description

Provides instant overview of project structure, current development phase, and key paths. Helps both new and existing team members quickly orient themselves.

## When to Use

- Starting work on the project
- After being away from the project
- Before making changes (called by /workflow-control)
- When lost in the codebase

**Trigger:** "show project structure", "where is X", "project overview"

## Instructions

Display project metadata, directory structure, current phase, recent changes, and key references.

## Examples

**Example 1:**
User: "show project context"
Assistant: [Displays structure, Phase 1 status, recent files]

**Example 2:**
User: "where do strategies live"
Assistant: [Shows src/strategy/ with description]
