# project-onboard

New member onboarding guide for crypto trading system.

## Description

Helps new developers quickly understand the project: goals, architecture, tech stack, and how to get started. Provides curated introduction without overwhelming with details.

## When to Use

- First time working on the project
- Onboarding new team members
- Refresher after long absence

**Trigger:** "onboard me", "introduce project", "I'm new here"

## Instructions

1. Project overview (from PROJECT_PLAN.md summary)
2. Tech stack and dependencies
3. Key architectural decisions
4. Development workflow
5. First tasks for newcomers
6. Where to get help

## Examples

User: "I'm new to this project, introduce it"
Assistant: [Shows project goals, tech stack, suggests reading START_HERE.md, lists beginner tasks]
