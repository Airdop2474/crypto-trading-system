## 📐 网格策略设计

### 核心逻辑

```python
class GridTradingStrategy:
    """
    网格交易策略
    
    原理：
    1. 在价格区间内设置多个网格
    2. 价格下跌到网格线时买入
    3. 价格上涨到网格线时卖出
    4. 赚取震荡市场的波动收益
    """
    
    def __init__(
        self,
        lower_price: float,    # 价格下界
        upper_price: float,    # 价格上界
        grid_count: int,       # 网格数量
        initial_capital: float # 初始资金
    ):
        # 计算网格间距
        self.grid_spacing = (upper_price - lower_price) / grid_count
        
        # 生成网格线
        self.grids = [
            lower_price + i * self.grid_spacing 
            for i in range(grid_count + 1)
        ]
        
        # 每个网格的投资金额
        self.amount_per_grid = initial_capital / grid_count
```

### 交易逻辑

```python
def on_bar(self, data, current_time):
    """
    网格交易逻辑
    
    买入规则：
    - 价格下穿网格线
    - 该网格未持仓
    - 有足够资金
    
    卖出规则：
    - 价格上穿网格线
    - 该网格有持仓
    """
    current_price = data.iloc[-1]['close']
    
    # 找到当前价格所在的网格
    current_grid = self.find_grid(current_price)
    
    # 检查是否下穿网格线（买入信号）
    if self.crossed_down(current_grid):
        if not self.is_grid_filled(current_grid):
            return "BUY"
    
    # 检查是否上穿网格线（卖出信号）
    if self.crossed_up(current_grid):
        if self.is_grid_filled(current_grid):
            return "SELL"
    
    return None
```

### 参数设置

**基础参数：**
- **价格区间：** 根据历史波动范围设置
- **网格数量：** 10-20个（可调整）
- **每格投资：** 总资金 / 网格数量

**优化参数：**
- **网格间距：** 根据市场波动调整
- **止损线：** 价格下界 - 10%
- **止盈线：** 价格上界 + 10%
