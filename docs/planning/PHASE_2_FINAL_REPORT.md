## 💡 技术亮点

### 1. 避免前视偏差的设计
```python
# 严格的时间顺序
for i in range(len(data)):
    current_bar = data.iloc[i]  # 当前K线
    historical_data = data.iloc[:i+1]  # 只能用到当前及之前
    signal = strategy.on_bar(historical_data)  # 生成信号
    
    if i < len(data) - 1:
        next_bar = data.iloc[i + 1]  # 下一根K线
        execution_price = next_bar["open"]  # 下一根开盘价成交
```

### 2. 完整的成本模型
```python
# 买入成本
execution_price = price * (1 + slippage)  # 滑点
cost_per_unit = execution_price * (1 + commission)  # 手续费

# 卖出成本
execution_price = price * (1 - slippage)  # 滑点
proceeds = quantity * execution_price * (1 - commission)  # 手续费
```

### 3. 代码静态分析
```python
# 检测危险模式
DANGEROUS_PATTERNS = [
    (r'\.iloc\[-1\]', '使用当前K线数据', 'high'),
    (r'\.shift\(-\d+\)', '使用未来数据', 'critical'),
]
```
